# Cal로리

1. 프로젝트 명 : Cal로리 <br>
Calculate + 칼로리<br>
3. 프로젝트 설명 <br>
사용자가 음식 사진을 업로드 하면 이미지 분류 모델이 음식을 분류하고 영양 정보를 알아낸다. 이를 바탕으로 사용자의 영양 상태를 모니터링 하고 부족한 영양 섭취를 보충해줄 음식 리스트를 제공하는 웹 서비스이다.
3. 사용될 데이터 <br>
